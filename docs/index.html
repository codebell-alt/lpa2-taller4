Características Principales

CRUD Completo

Gestión completa de Usuarios, Canciones y Favoritos con operaciones Create, Read, Update y Delete.

FastAPI PostgreSQL

Frontend Moderno

Interfaz responsive construida con Next.js, React 19 y Tailwind CSS con gestión de estado avanzada.

Next.js 16 TypeScript

Pruebas Automatizadas

Suite completa de pruebas con Vitest para frontend y pytest para validación de contratos API.

Vitest pytest

Paginación y Filtros

Sistema avanzado de paginación y filtros por género, artista, año y búsqueda avanzada.

Validaciones Robustas

Validación en tiempo real con Zod y React Hook Form, más validaciones del lado del servidor con Pydantic.

DevOps

Pre-commits con Husky, Conventional Commits, y documentación completa con MkDocs.

Endpoints de la API

Usuarios

GET /api/usuarios/

Listar usuarios con paginación

Query params: skip, limit

{
  "items": [...],
  "total": 100,
  "page": 1,
  "size": 100,
  "pages": 1
}
POST /api/usuarios/

Crear nuevo usuario

{
  "nombre": "Juan Pérez",
  "correo": "juan@example.com"
}
GET /api/usuarios/{usuario_id}

Obtener usuario por ID

PUT /api/usuarios/{usuario_id}

Actualizar usuario

DELETE /api/usuarios/{usuario_id}

Eliminar usuario

Canciones

GET /api/canciones/

Listar canciones con paginación y filtros

Query params: skip, limit, genero, artista, año_min, año_max

POST /api/canciones/

Crear nueva canción

{
  "titulo": "Bohemian Rhapsody",
  "artista": "Queen",
  "album": "A Night at the Opera",
  "duracion": 354,
  "año": 1975,
  "genero": "Rock"
}
GET /api/canciones/buscar/avanzada

Búsqueda avanzada de canciones

Query params: titulo, artista, genero, año

GET /api/canciones/generos/lista

Listar todos los géneros disponibles

Favoritos

GET /api/favoritos/

Listar favoritos con paginación

POST /api/favoritos/

Marcar canción como favorita

{
  "id_usuario": 1,
  "id_cancion": 5
}
GET /api/favoritos/usuario/{usuario_id}

Obtener favoritos de un usuario

GET /api/favoritos/verificar/{usuario_id}/{cancion_id}

Verificar si una canción es favorita

Utilidades

GET /health

Health check de la API

GET /stats

Estadísticas generales de la base de datos

Instalación y Uso

1. Clonar repositorio

git clone https://github.com/codebell-alt/lpa2-taller4.git
cd lpa2-taller4

2. Instalar dependencias

pnpm install
cd frontend && pnpm install

3. Ejecutar frontend

cd frontend
pnpm dev

4. Ejecutar pruebas

# Frontend
cd frontend && pnpm test

# API
cd tests_api && pytest -v
